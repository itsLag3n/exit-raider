## How to run

### Just run
- `start.bat` on **Windows**

### Or follow these steps
Clone the project

```bash
  git clone https://github.com/itsLag3n/exit-raider.git
```

Go to the project directory

```bash
  cd exit-raider
```

Install requirements

```bash
  # Linux/macOS
  pip3 install requests

  # Windows
  pip install requests
```

Start the tool

```bash
  # Linux/macOS
  python3 main.py

  # Windows
  py main.py
```
